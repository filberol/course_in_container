# Инфраструктура и DevOps

Методические материалы для учебной дисциплины **«Инфраструктура и DevOps»**, посвящённой современным практикам управления разработкой, доставкой и эксплуатацией программного обеспечения (DevOps, CI/CD, контейнеризация, Kubernetes, IaC, SRE).

Репозиторий содержит лекционные материалы, лабораторные работы и вспомогательные ресурсы, используемые в рамках курса.

---

## Цели курса

- Сформировать целостное понимание DevOps-культуры и принципов
- Познакомить с жизненным циклом разработки ПО (SDLC)
- Освоить контейнеризацию и оркестрацию (Docker, Kubernetes)
- Научиться выстраивать CI/CD-пайплайны
- Изучить подход Infrastructure as Code (Terraform, Ansible)
- Рассмотреть мониторинг, логирование, DevSecOps и основы SRE

## Для кого этот курс

- студенты IT-направлений (ПО, ИБ, ИТ-инфраструктура);
- начинающие DevOps / SRE-инженеры;
- разработчики, желающие лучше понимать процессы доставки и эксплуатации.
