# Инфраструктура ПО и инструменты управления разработкой

Методические материалы и наработки для курса. Приветствуется любое участие и вклад в разработку курса.

## Структура репозитория

- `Лекции/` — конспекты 16 лекций и план слайдов (`_Список слайдов.md`)
- `Лабораторные/` — задания лабораторных работ
- `slides/` — исходники презентаций лекций ([Slidev](https://sli.dev)) и фирменная тема курса
- `Рабочая программа дисциплины.md`, `Вопросы к экзамену.md` — методические документы
- `Карта курса.canvas` — карта курса (Obsidian Canvas)

## Презентации лекций

Каждой лекции соответствует колода `slides/lectures/NN.md` с заметками докладчика —
полным текстом выступления. Оформление задаёт локальная тема `slides/theme`
(палитра по брендбуку ИТМО, диаграммы Mermaid в едином стиле).

Локальная работа (Node.js ≥ 24):

```bash
cd slides
npm ci
npm run dev -- lectures/01.md          # живой просмотр с заметками докладчика
npm run export -- lectures/01.md --output dist/lecture-01.pdf   # PDF
```

Шаблон оформления — `slides/lectures/00-template.md`, правила подготовки колод —
`slides/AGENT_GUIDE.md`.

### Сборка в CI

Workflow `.github/workflows/slides.yml` при изменениях в `slides/**` экспортирует все
16 колод в PDF и публикует их артефактами: `lecture-NN` по отдельности и `all-lectures`
одним архивом (вкладка Actions → выбранный запуск → Artifacts).
